# Aloha Pot

## Description

[ALOHA 2](https://aloha-2.github.io/) robot lifting a pot off the work table.  This benchmark exercises [GJK](https://en.wikipedia.org/wiki/Gilbert%E2%80%93Johnson%E2%80%93Keerthi_distance_algorithm) in a simple, single-object manipulation scenario.

### aloha_pot

| Property | Value |
|----------|-------|
| Bodies | 26 |
| DoFs | 23 |
| Actuators | 14 |
| Geoms | 204 |
| Timestep | 0.002s |
| Solver | Newton |
| Friction | Elliptic |
| Integrator | Euler |
| Matrix Format | Dense |
